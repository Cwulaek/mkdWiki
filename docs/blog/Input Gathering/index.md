# Input Gathering

sc本身的输入功能非常鸡肋，只能检测是否按下某个键。

至少做到鼠标侦测（当角色被点击）那样吧。

需要说明的是，我们主要以[URM](../URM/index.md)作为底层。

## 键盘侦测

为了便捷地动态增减要侦测的按键，我们要用列表把它们存起来。  

### 包装按键

我们可以简单设想一下我们要实现的：检测点击、抬起和长按。  
让我们先把对于每种按键都要进行一遍的`如果<()按下>...否则...`化简一下。

我讲下想法，免得这步有些跳跃。

首先，用列表存储要侦测的按键种类，并将按键命名，以方便游戏中自定义按键。  
然后，在每次更新时调用按键侦测。
