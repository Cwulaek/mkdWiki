# 面

##### 我们的脸，我的我们，他们的所见。

-----

宇宙是面，世界是体，这些是二和三。

什么是四？我不知道，直到五棋。

以线表示面，以新线表示树。  
树是线的立方，树在新面中。  
这是我的构思。

大火，中精。当时使用画布，初生之勇。

话回前言。卡战之为回归时首作，其时策多，可无能为力使其不进。

初生之勇使我研究语言机理，毁卡战由此而致。

面是球之展开。后月，我作球，球之进洞。

初次伪立方就成功，我使用我们的能力增强。

以号表块，表示块的面。  
记载方向和距离，增加受力。  
大火，中精。

此时我们已上千，甩人以万里。

图章已为我所熟，URM构建渐渐形成。

URM又为更渲控。

更师更新，渲师渲染，控师运算。

当时无控但有广，由线程而引入。  
事之不知，之之运算于更，慢而繁。

当时又少封装。间于球余，魔方出，其六面。

夫其，不全转而劣，自动而有优。时五棋与六面封装略，球则少。

然面中有线，线现为象，后文矣。

皆知台为面，却我不解。众好美而独陋不止一时，常劣我者仍优以美。

台跳社区皆易火，更易制。三之大作沉极速，台跳后，唯美而精者久。

三服。

三技疏，多尝试新坑，而未填又十。有台跳形地城我世，有暮森于皇家守军，有火红口妖，有音线传说之下及三符文。

当时我们是面，面共一线，合为UR而无M。

面有线而无点。

我们一心不二用，专而不可扩，易之则为误。总有半年，多困于不知所写。我们要繁，我要全：

一止于UR，二止于效，三止于我。

中考后，高一半年又疫情网课，多游而无写。及终，念爪哇之事件类，开事件架构，做雏之新口妖，有核心。

2023寒，真觉醒之年初。
