卡自小引我念，牌至大亦有趣。
年方六七，有赛尔之船时优，有卡之可战。遂收多卡以收图鉴而游余乐。又有课外积分卡，百而未花，实无趣之可买，惟一指尖陀螺。时初试月圆也，亦初。
年至十，游UNO及三国杀于校，培卡牌之趣也。
年至十二，自制牌。有富城、炉传、三国又余。恰归来时，仿月圆而作，望精。后为减代而制解析，然速慢且错。于校，与友以炉传比试，多长牌略，乃归月圆，通也。
年十五，事件出，大作可为。年十七，半年由轮为作，制地传。上呼卡战，下从新作，旁称大轮。
是日二五年初二月，讲轮之术于此。
轮子由用牌法、抽牌法、弃牌法、撕牌法、伤害法、效果法、监听法、触发法、制牌法、历史法、全局法。
用牌者，察其能不，问其能不，而行之，告于其，终而归。
抽牌者，先问，套移牌之术复问，终得牌，有两告于其。
弃牌与撕牌者，法似而渲不同与抽牌，略也。
伤害者，有来源，先问以改伤，再实告于其。
效果者，监听为主，触发为辅，而功能在辅，仅施时在主。
监听者，一事不同法，以参定之，似面向对象也。
触发者，先加而有触，换头以之监听，得触也。
制牌者，原无以添为邻，或有而复之亦为临，或均不。
